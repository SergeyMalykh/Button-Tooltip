Part I - Front End

Objective

Create a single page Angular 4+ app with simple routing that has one state/view. In it, place two buttons that when clicked; show a tooltip above the button with some text in it.
When button A is clicked, if button Bâ€™s tooltip is open it should close and vice versa - only one tooltip should be visible at a time. Either tooltip should close when clicked anywhere outside of it, but remain open if clicked inside. Either tooltip should close when ESC key is pressed.

Implementation

  1. Tooltip functionality should be written as a reusable component (Its activated by attribyte 'appCustomButtonTooltip'. See CustomButtonTooltipDirective)
  2. You are encouraged to use a starter/seed project to save time scaffolding (Used Angular Command Line Interface:
    ng new tooltip
    ng serve --open
    ng generate component tooltip-test
    ng generate directive common/customButtonTooltip)
  3. You can organize the files/folders in any way
  4. The reviewer should be able to run your solution with a simple npm command
  5. You should submit your solution as a github repository link
